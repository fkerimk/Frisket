# Frisket

Small desktop tool for perspective cropping, creating seamless textures, and applying retro effects.

<img width="774" height="480" alt="frisket" src="https://github.com/user-attachments/assets/dc9dcaf2-aaec-423b-bd46-006971fde648" />

## Run

```bash
dotnet run
```
You can open a file simply by dragging and dropping it. You can also enter the file path as the first argument.

You can reset the values by right-clicking, or adjust them using the mouse wheel.

You can make precise adjustments independently of the snap function by pressing the Alt key.

## Dependencies

- `Raylib-cs` - https://github.com/raysan5/raylib/blob/master/LICENSE
- `rlImgui-cs` - https://github.com/raylib-extras/rlImGui-cs/blob/main/LICENSE
- `OpenCvSharp4` - https://github.com/shimat/opencvsharp/blob/main/LICENSE
- `NativeFiledialogNET` - https://github.com/atomsk-0/NativeFiledialogNET/blob/master/LICENSE
